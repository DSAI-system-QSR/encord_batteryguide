🟡 FANUC (화낙) 배터리 교체 절차

1준비물품 확인
⚠️ 중요: 맨손이나 면 장갑 착용 시 정전기로 인한 기판 손상 위험이 있습니다.
  • 절연 장갑 (정전기 방지용)
  • (-) 주먹 드라이버
  • 1.5V D타입 배터리 4개
2안전 준비
⚠️ 중요: 전원 OFF 시 원점 소실 위험이 있습니다.
  • 전원 ON 상태 확인
  • OFF 모드 선택
  • 안전 도어 OPEN
  • 필요시 비상정지 ON
3배터리 위치 확인
  • (메인)제어전원 하단의 배터리 단자 박스 확인
  • 52호기는 측면 배터리 단자 확인
  • 전해액 누수 등 오염 상태 점검 및 청소
4배터리 교체
  • 드라이버로 커버 개방
  • 극성(+,-)에 주의하여 배터리 4개 교체
5완료 확인
  • 비상정지 및 OFF 버튼 해제
  • 배터리 알람 해제 확인
  • 전원 OFF 후 재투입하여 정상 작동 확인

🔵 SUMITOMO (스미토모) 배터리 교체 절차

1준비물품 확인
⚠️ 중요: 정전기 방지를 위해 반드시 절연 장갑을 착용하세요.

정품 배터리 사용 시:

  • 절연 장갑 (정전기 방지용)
  • SAFT LS26500 3.6V 1개
  • 커넥터 미선택 시 배선연결 도구

일반 배터리 사용 시:

  • 절연 장갑 (정전기 방지용)
  • 1.5V D타입용 배터리 홀더
  • 1.5V D타입 배터리 3개
  • 다목적 가위, 얇은 케이블 타이 2개
2안전 준비
⚠️ 중요: 전원 OFF 시 원점 소실 위험이 있습니다.
  • 전원 ON 상태 확인
  • OFF 모드 선택
  • 모터 OFF 설정
  • 안전 도어 OPEN
  • 필요시 비상정지 ON
3기존 배터리 제거
  • 제어부 패널 오픈
  • 기존 배터리 상태 확인
  • 커넥터 파손 및 기판 접촉 주의하여 커넥터 탈거
  • 배터리 분리
4새 배터리 설치
  • 새 배터리 연결
  • 커넥터를 원래 위치에 복구
  • 필요시 케이블 타이로 고정
5완료 확인
  • 이상 해제(리셋) 실행
  • 정상 리셋 확인
  • 모터 ON 및 동작 체크

🚨 즉시 교체가 필요한 상황

FANUC의 경우:

  • 배터리 알람 APC전압 낮음 이상 표시
  • 배터리 이상 표시 발생

SUMITOMO의 경우:

  • 전원 OFF 후 재투입 시 각 축 서보이상 5번 알람 발생
  • 설비 이상 없이 전원 OFF 후 5번 알람 발생
📋 교체 주기: 매년 1회 또는 2년에 1회 권장
🔍 정기 점검: 전해액 누수 및 커넥터 상태 확인
